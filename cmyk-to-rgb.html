Frequently Asked Questions - CMYK RGB FAQ

What is the difference between CMYK and RGB color modes?

CMYK is a subtractive color model used for printing. RGB is an additive color model used for digital screens. CMYK uses Cyan, Magenta, Yellow, Black ink while RGB uses Red, Green, Blue light. This is why same color looks different in print vs on screen.

Can I convert RGB back to CMYK using this tool?

Currently this tool converts CMYK to RGB only. RGB to CMYK conversion is complex because RGB has wider color range. For print design, always design in CMYK mode in Photoshop or Illustrator for accurate results.
